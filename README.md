### 2. Which text will print first before from below code ?
```
void main() async{
 await recall();
  print('y');
}
recall(){
  Future.delayed(Duration(seconds:4),(){print('x');});
}

```
Ans: y will print first and then x will print. because dart support asynchronous Programming
